# 🐍 CSE110: Programming Language I (Summer - 2021) 🎓

Welcome to the ultimate repository for **CSE110: Programming Language I**! 🚀 

This repository contains all the solutions for the **BRAC University** CSE110 Lab Assignments from the **Summer 2022** semester. Whether you are revising for your finals, stuck on a specific problem, or just exploring introductory Python concepts, you've come to the right place! 💡

All solutions are neatly organized into Jupyter Notebook (`.ipynb`) files, meaning you get all the solutions for a single assignment in one convenient, easy-to-read file. 

---

## 🌟 Highlights & Features

* **Complete Solutions:** 100% of the lab tasks are solved and tested.
* **Modern Python:** All code is written in **Python 3.9**, utilizing modern best practices and clean syntax.
* **Jupyter Notebooks:** Code, outputs, and explanations are bundled together in `.ipynb` files for an interactive learning experience.
* **Beginner Friendly:** The code is heavily commented to help you understand the *why* and *how* behind the logic.
